# Setup Guide

이 리포지토리는 RL 학습 트랙의 뼈대만 제공합니다. 실제 구현은 모두 TODO 상태입니다.

## Environment
- Python 3.9+
- 권장: `python -m venv .venv && source .venv/bin/activate`
- 설치: `pip install -e .`

## Structure
- Atari DQN: `projects/atari_dqn/`
- Pykachu volleyball: `projects/pykachu_volleyball/`
- Omok: `projects/omok/`
